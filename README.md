# データ分析

金沢学院大学 情報工学部「データ分析」（担当: 後藤弘光）の演習教材リポジトリです。

対象: 情報工学部 情報工学科 2 年次

---

## 授業概要

Python を用いてデータの収集・加工・可視化・統計分析を行う演習科目です。前半ではデータの取り込みから整形・集計・可視化までの一連のデータエンジニアリングを、後半では回帰分析・統計検定・クラスタ分析・分散分析の 4 つの統計分析手法を学びます。

本科目のキーメッセージ: 「描いてから、書く」 — データ分析は、いきなりコードを書くものではありません。まず分析の骨組み（手順）を日本語で設計し、それを一つずつ Python コードに翻訳していきます。生成 AI によってコード生成が容易になった現在、分析者に求められる価値は「問いの設定」と「結論の言語化」へ移行しており、この設計力を反復トレーニングで養うことを本科目の中心に据えています。

## 到達目標

- 与えられたデータに対して、収集・加工・可視化の一連の前処理を Python で実行できる
- 分析課題に対して適切な統計手法（回帰・検定・クラスタ・分散分析）を選択し、Python で実装できる
- 分析結果を解釈し、課題に対する結論を言語化できる

## シラバス

前半10回で統計分析に必要なデータの取り扱い基盤（収集・整形・結合・可視化）を固め、後半4回でその基盤の上に統計分析手法を積み上げる構成です。

## 教材構成

各回のフォルダには以下が含まれます。

```
da_XX/
├── da_XX.ipynb    # 要点まとめ＋演習（Jupyter Notebook）
└── data/          # 演習用データ（ある場合）
```

各ノートブックは以下の構成で統一されています。

1. 要点まとめ — 本時のテーマに関する理論解説とデモコード
2. 演習 — 架空のケース（課題）を題材にした課題解決型の演習。背景から分析の問いを立て、仮説を定義し、骨組みコメントでコードを設計してから実装に進むプロジェクトベース学習の形式を採用しています

骨組みコードセルには授業中に日本語のコメント行（`# 1. …, # 2. …`）で手順を書き出し、その下にコードを追記していく「設計 → 翻訳」の流れで取り組みます。

## 環境構築（初回のみ）

本科目では、自分の PC にローカル環境を構築して JupyterLab 上で作業します。仮想環境の作成・パッケージ管理・トラブルシュートといった環境構築の経験は、実務や研究で不可欠な基礎スキルです。

### 1. Anaconda のインストール

以下のサイトを参考にインストールしてください。

- [Python 環境構築ガイド](https://www.python.jp/install/install.html)
- [Anaconda](https://www.anaconda.com/products/distribution)

### 2. 仮想環境の作成

```bash
conda create -n da_env python=3.11 -y
```

### 3. 仮想環境の有効化

```bash
conda activate da_env
```

プロンプトに `(da_env)` と表示されれば OK です。

### 4. 必要パッケージのインストール

```bash
pip install -r requirements.txt
```

---

## 課題の取り組み方

### 1. 仮想環境を有効化する

```bash
conda activate da_env
```

### 2. JupyterLab を起動する

```bash
jupyter lab
```

### 3. 課題に取り組む

各回のフォルダ内にある `da_XX.ipynb` を開き、要点まとめを読んだ上で演習に取り組みます。骨組みコードセルに手順のコメントを書いてから、コードを追記してください。

### 4. 作業を終えたら無効化する

```bash
conda deactivate
```

---

## 仮想環境の削除（任意）

すべての授業が終了し環境が不要になった場合は、以下で削除できます。

```bash
conda remove --name da_env --all
```
